# Apex-Drift
APEX DRIFT is a futuristic React karting arena site with 3 pages: Home (hero, track modals, membership upsells, animations), Booking (live pricing wizard, popups), Checkout (live card UI, validation, success modal). Stack: React 18, Vite 5, Tailwind CSS, Framer Motion, React Router, Lucide React, JSX only.
